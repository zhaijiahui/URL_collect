# URL_collect
用于收集网站URL的脚本


#---------------------------

#URL collect by zhaijiahui

#---------------------------


-u  Input your domain

-d  Crawl depth

-o  Save result

-s  Prevent requests too fast

Usage: 
	get_url.py -u http://www.target.com/ -d 2
	get_url.py -u http://www.target.com/ -d 2 -s 2 -o
